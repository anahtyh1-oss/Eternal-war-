# Eternal War: Political Simulation Mobile

A strategic political simulation mobile game built with Godot Engine, featuring dynamic NPC agents, diplomatic negotiations, and complex legal frameworks.

## Project Structure

```
PoliticalSimulationMobile/
├── scripts/           # GDScript files for game logic (Kimi)
├── data/              # JSON data files (Qwen)
├── assets/            # Game assets - sprites, audio, UI (Mistral & Grok)
├── docs/              # Game design and documentation (Claude)
├── tests/             # Test reports and bug tracking (Perplexity)
└── audio/             # Audio assets and animations (Grok)
```

## Features

- 🎮 Grid-based world with multiple factions
- 👥 Dynamic NPC agents with autonomous behavior
- ⚖️ Legal framework system
- 📦 Inventory management system
- ⏰ Time simulation with in-game events
- 📱 Mobile-optimized UI

## Getting Started

1. Clone this repository
2. Open the project in Godot Engine (version 4.0+)
3. Run the main scene to start the game
