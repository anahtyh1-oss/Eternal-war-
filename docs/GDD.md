# Game Design Document - Eternal War: Political Simulation Mobile

## Overview
Eternal War is a political simulation mobile game where players engage in diplomatic negotiations, manage resources, and influence the fate of a medieval fantasy world.

## Core Mechanics
- Political Diplomacy
- Resource Management
- NPC Agent Simulation
- Time-based World Events
- Legal Framework System

## Target Platform
- iOS and Android

## Art Style
- 2D Pixel Art
- Grid-based Isometric View
