# Game Rules - Eternal War

## Basic Rules
1. Each player controls a faction with its own territory and resources
2. NPCs act autonomously based on their allegiances and motivations
3. Laws can be enacted to influence NPC behavior
4. Time progresses in-game, triggering events and state changes

## Victory Conditions
- Political Dominance: Control majority of the world
- Economic Victory: Accumulate wealth and trade routes
- Diplomatic Victory: Ally with all major factions

## Resource System
- Gold: Currency for trade
- Influence: Political capital
- Supplies: For military/infrastructure
