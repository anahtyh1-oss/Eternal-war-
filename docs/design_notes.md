# Design Notes - Eternal War

## Technical Considerations
- Use Godot Engine for cross-platform development
- JSON-based data system for easy modding
- Grid-based pathfinding for NPC movement

## Art Direction
- Inspired by classic tactical RPGs
- Color palette: Medieval fantasy theme
- UI designed for mobile touch controls

## Future Features
- Procedurally generated world
- Multiplayer diplomacy
- Custom law creation system
